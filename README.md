# Simple API with Golang and GORM

This is a simple RESTful API built with Golang and GORM (Go Object-Relational Mapping) for managing a collection of items. The API supports basic CRUD (Create, Read, Update, Delete) operations.

## Features

- Create a new item
- Retrieve all items
- Retrieve a single item by ID
- Update an existing item
- Delete an item

## Prerequisites

- Go (version 1.16 or higher)
- GORM library
- A database (e.g., SQLite, MySQL, PostgreSQL)
